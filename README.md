## Git 管理

本项目使用 Git 进行版本管理。

```bash
# 克隆后首次操作
git config --global init.defaultBranch main
git checkout -b develop

# 提交规范
feat: 新功能
fix: 修复bug
docs: 文档更新
style: 代码格式
refactor: 重构
test: 测试
chore: 构建/工具
```
